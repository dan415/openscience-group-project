Installation
-------------
The openscience-grupo project can be easily installed by following these steps:

1. Clone the GitHub repository:

   .. code-block:: bash

      git clone https://github.com/dan415/openscience-grupo.git

2. Navigate to the project directory:

   .. code-block:: bash

      cd openscience-grupo

3. Set up the virtual environment:

   .. code-block:: bash

      python -m venv venv
      source venv/bin/activate

4. Install the dependencies:

   .. code-block:: bash

      pip install -r requirements.txt

Usage
-------------
To use openscience-grupo, follow these steps:

1. Activate the virtual environment:

   .. code-block:: bash

      source venv/bin/activate

2. Run the main script:

   .. code-block:: bash

      python main.py

3. Follow the prompts and provide the necessary inputs to interact with the project.

Queries
-------------
The openscience-grupo project provides a set of predefined SPARQL queries to retrieve specific information from the dataset. These queries can be used to perform advanced searches, analyze data, and extract meaningful insights. The queries can be found in the `queries` directory of the project.
